# 🚀 Universal Video Downloader Pro

**Universal Video Downloader Pro** is a powerful, smart, and feature-rich desktop application designed to download videos and audio from various platforms with ease. Built with a clean UI and robust backend behavior, it ensures reliable downloads with cookie support and complete metadata extraction.

## 🔥 Key Features

*   **Multi-Platform Support:** Seamlessly download from **YouTube, Facebook, TikTok, Playlists**, and convert to **MP3**.
*   **Smart Metadata & Thumbnails:** Support for embedding metadata, writing thumbnails, and previewing metadata before downloading.
*   **Advanced Cookie Support:** Pass your browser cookies to download private or restricted videos safely.
*   **Flexible Download Settings:**
    *   Choose your preferred video quality (up to the best available).
    *   Custom filename formatting (e.g., *Title + ID*).
    *   Download playlists and subtitles effortlessly.
    *   Skip already downloaded files to save time and data.
    *   Speed limit control to manage your network bandwidth.
*   **URL Analysis:** Analyze links before starting the download to preview formats.
*   **Dependency Checker:** Built-in tool to check system requirements like Python, yt-dlp, and FFmpeg directly from the UI.

---

## 🛠️ System Dependencies

The tool relies on the following backend components for its core operations:
*   **Python:** (Optimized for Python 3.14+)
*   **yt-dlp:** (Handles core extraction and downloading)
*   **FFmpeg:** (Required for merging high-quality video/audio tracks and conversion)
